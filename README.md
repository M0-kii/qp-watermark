# qp-watermark

## What Is qp-watermark?

qp-watermark is a ridiculously easy script that I made in 30min 😅. All this does is simply draw a logo over your screen (similar to a watermark). I also added a simple neon animation to it. plus when you enter a vehicle it moves to top right of screen. If there is actual interest in this I will update it and add a toggle watermark command. Otherwise, enjoy!

## How To Change Logo

- Simply put the image inside `html` folder
- you should name the image `logo.png` or its not gonna work

## How Can I Change The Position, Size, ETC..

- Edit the codes in the `index.html` file
- I will not provide support for any edits made to the files.
